# Security Policy

## Supported Versions

Give as much information as you can.

| Version | Supported          |
| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| 5.0.x   | :x:                |
| 4.0.x   | :white_check_mark: |
| < 4.0   | :x:                |

## Reporting a Vulnerability

Raise an [issue](https://github.com/yatesco/dev-docker/issues) and it will be dealt with with urgency.
